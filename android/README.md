# Android (Field App)

This folder contains a minimal Android app scaffold (Jetpack Compose) aligned with `plans/screen list.lua`.

## Running (Android Studio)
1. Open `android/` as a project.
2. Ensure Android SDK platform `android-34` is installed (it is present on this machine).
3. Run the `app` configuration.

## Backend / API
- Web/API lives in the repo root (PHP).
- API base for emulator is typically `http://10.0.2.2:8000/api/` (host loopback).
